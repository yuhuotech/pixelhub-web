现代化图床系统

PixelHub 是一个开源、美观且功能强大的图片托管解决方案。支持多存储后端，提供极致的上传体验和优雅的图片管理功能。

立即开始 Star on GitHub

精心打造的核心功能

每一个细节都经过深思熟虑，只为提供最流畅的图片管理体验。

多种上传方式

支持拖拽上传、点击上传。更支持 Ctrl+V 粘贴上传和 URL 网络图片自动抓取上传。

多存储后端

无缝集成腾讯云 COS、阿里云 OSS、GitHub、Gitee 和本地存储，按需切换。

实时搜索

毫秒级响应的实时搜索功能，让您在海量图片中瞬间找到所需内容。

时间轴视图

按年月自动分组的时间轴视图，让图片管理井井有条，回忆触手可及。

安全可靠

支持访问密码保护，回收站软删除机制，防止误删，数据安全有保障。

深色模式

完美适配的深色模式，呵护您的双眼，提供沉浸式的使用体验。

广泛的存储支持

无论您倾向于云服务还是私有部署，PixelHub 都能满足。

准备好部署了吗？

仅需几行命令，即可拥有属于您的现代化图床。

1

克隆仓库

下载源码到本地

2

安装依赖

使用 pnpm 或 npm 安装

3

启动服务

开始您的图床之旅

查看完整部署文档

git clone https://github.com/yuhuotech/pixelhub.git

cd pixelhub

pnpm install

# 配置 .env 文件...

pnpm dev
